1. Setup Environment

1.1. Install Rustup

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
  • Rustup - Version Management
rustup --version
  • Cargo - Package Management
cargo --version
  • Rustc - Rust Compiler
rustc --version

1.2. Move Programming Language

  • Clone Move Language Repository
git clone https://github.com/move-language/move
  • Setup & Restart Terminal
cd move && ./scripts/dev_setup.sh -ypt
  • Install Move CLi
cargo install --path language/tools/move-cli
  • Move Tools
cd ~/move/language/tools && tree -L 1 .

.
├── README.md
├── move-bytecode-utils
├── move-bytecode-viewer
├── move-cli
├── move-coverage
├── move-disassembler
├── move-explain
├── move-package
├── move-resource-viewer
├── move-unit-test
└── read-write-set

1.3. Install SUI Move

  • Sui Documentation
  • Sui explorer
  • Installs the sui package from the devnet branch of the MystenLabs GitHub repository.
  • It uses the cargo package manager which is used for Rust programming language.
cargo install --locked --git https://github.com/MystenLabs/sui.git --branch devnet sui
  • Version
sui --version
  • Install the Move analyzer for VS CODE
  • Add Compotibility for Sui Style Wallet
cargo install --git https://github.com/move-language/move move-analyzer --features "address32"
  • Installs the move-analyzer package from the sui-move branch of the Move language GitHub repository with the "address32" feature.
cargo install --git https://github.com/move-language/move move-analyzer --branch sui-move --features "address32"

2. Sui CLI

2.1. Sui version

  • This command outputs the installed version of sui.
sui --version

2.2. Genesis

  • Localnet - Devnet - Testnet - Mainnet
  • Initializes the SUI environment by creating a genesis block & accounts.
sui genesis
  • Displays the directory structure of the .sui directory.
tree ~/.sui

2.3. Sui Console

  • SUI console interface.
sui console

2.4. Address Menager

2.4.1. Addresses

  • Displays a list of addresses managed by the SUI client.
sui client addresses

2.4.2. Active address

  • Shows the currently active address in the SUI client.
sui client active-address

2.4.3. Change Active address

  • Switches the active address in the SUI client to the specified address.
sui client switch --address 0x901c6c17666e8844552abe72132ca7075c4bfc087f189376d66b8fd78606b2f3

2.4.4. Create New Address

  • Generates a new address using the secp256k1 encryption method.
sui client new-address secp256k1
  • Generates a new address using the secp256r1 encryption method.
sui client new-address secp256r1
  • Generates a new address using the ed25519 encryption method.
sui client new-address ed25519

2.4.5. Gas

  • Retrieves the current gas balance of the active address
sui client gas
  • Retrieves the gas balance of a specified address.
sui client gas 0x3e3a5708d6eb4de4ff7c172ac3c8ca811eb49b5416855a1eb9689e0f20263e0c

2.4.6. Object

  • Retrieves details of an object using its ID.
sui client object 0xcef366d8c7f8ae0d7b5dca59565acf62e410600cd219c0040412f5ba39891e25

2.4.7. Objects

  • Lists all objects owned by the active address.
sui client objects

2.5. Token manager

2.5.1. Transfer Token

  • Transfers a specified object to a given address, using optional gas parameters.
sui client transfer --to ${address} --object-id ${object_id} --gas(optional) ${object_id} --gas-budget ${number}

2.5.2. Split

  • Splits a coin into specified number of pieces.
sui client split-coin --coin-id ${object_id} --count ${piece} --gas ${object_id} --gas-budget ${number}
  • Splits a coin into specific amounts.
sui client split-coin --coin-id ${object_id} --amounts ${piece} ${piece} --gas ${object_id} --gas-budget ${number}

2.5.3. Merge

  • Merges two coins into a primary coin
sui client merge-coin --primary-coin ${object_id} --coin-to-merge ${object_id_merge} --gas ${object_id} --gas-budget ${number}

2.6. Environment

2.6.1. Switch to Devnet

  • Lists the available environments in the SUI client.
sui client envs
  • Switches the SUI client to the 'devnet' environment.
sui client switch --env devnet

2.6.2. Addition environment

  • add this to ~/.sui/suiconfig/client.yaml
  - alias: mainnet
    rpc: "https://fullnode.mainnet.sui.io:443"
    ws: ~

2.6.3. Faucet Gas

  • Creates a new account on Devnet using the ed25519 encryption method.
sui client new-address ed25519
